Project: 
  This project implements a Pick-and-Place system using a Panda robot (panda_gz_moveit2) in a ROS2 and Gazebo simulation environment.
  The goal is to make the robot pick up a selected object, such as an ingredient, and place it into a pot. 
  To achieve this, ROS2 Humble, MoveIt2, Gazebo Fortress, and the Panda robot simulation package were used.

AI Usage:
  - Understanding ROS2, MoveIt2, Gazebo, and controller concepts
  - Debugging build and execution errors
  - Organizing the structure of the custom Pick-and-Place node
